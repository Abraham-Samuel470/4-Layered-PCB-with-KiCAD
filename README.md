# 🚀 ESP32-S3-WROOM-1 Custom 4-Layer PCB Design

## 📌 Overview
This project showcases the design and development of a **4-layer custom PCB** based on the **ESP32-S3-WROOM-1** module.  
The goal of this project was to gain hands-on experience in **advanced PCB design**, including multilayer stack-up, routing, and power integrity.

---

## 🧠 Inspiration & Learning
This project was inspired by the tutorial from Morten (Made by Morten YouTube channel).  
I followed his workflow to understand real-world PCB design practices and implemented the design independently to strengthen my concepts.

---

## 🛠️ Tools & Technologies
- KiCad (PCB Design & Schematic)
- ESP32-S3-WROOM-1 (Microcontroller Module)
- Multilayer PCB Stackup (4-Layer Design)

---

## 🧩 PCB Stackup
- **Layer 1 (Top):** Signal + Components  
- **Layer 2:** Ground Plane (GND)  
- **Layer 3:** Power Plane (VCC)  
- **Layer 4 (Bottom):** Signal Routing  

---

## ⚙️ Key Features
- ✅ 4-Layer PCB Design  
- ✅ Proper Ground & Power Plane Separation  
- ✅ Optimized Routing for Signal Integrity  
- ✅ Compact ESP32-Based System Design  
- ✅ USB / Peripheral Interface Considerations
